# UI Direction (Applied)

## Visual Theme
- Glassmorphism shell with aurora background.
- Strong hero with animated DNA orb.
- Gradient accents and collectible vibe without dark-mode dependency.

## Result Experience
- DNA crest presentation instead of plain square icon.
- Impact score card + collection count card.
- Achievement badges with tiers and impact points.
- Progress-to-next rendered as visual tracks.

## Codex Discovery
- Dedicated page (`codex.html`) instead of inline section.
- Main CTA buttons: `Explore DNA Codex` and `See This Type in Codex`.
- Codex detail panel with live character preview.

## Rate-Limit UX
- PAT is primary path for reliable live analysis.
- Session PAT memory (tab scope only).
- Better warning copy with clear action.
